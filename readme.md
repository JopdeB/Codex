# Mila's Moonlit Garden

An interactive web-based storybook that brings a moonlit bedtime adventure to life with gentle animations, branching paths, and playful activities for young readers.

## Getting started

1. Open `index.html` in your favourite browser. No build step is required.
2. Click **Start the Story** to accompany Mila and her firefly friends.
3. Use the **Read Along** button for highlighted narration, solve the activities on each page, and explore both story paths to hear every version of the lullaby.

## Project structure

- `index.html` – Page layout and storybook container.
- `styles.css` – Whimsical styling, responsive layout, and animations.
- `script.js` – Story data, navigation logic, read-along highlighting, and interactive activities.

Enjoy weaving your own version of the moonlight lullaby!
